# liveprinter-core

Core library for LivePrinter with programmatic GCode renderer. Can render JavaScript code to GCode for use in Marlin-like printers like Ultimakers and Prusas and more.

see [FUNCTIONS.md](./FUNCTIONS.md) for a list of all functions and documentation.

see https://github.com/pixelpusher/LivePrinter for the original library and usage

see https://github.com/pixelpusher/liveprinter2 for a current example of the LiveCoding editor.
